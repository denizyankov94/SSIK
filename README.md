**Важни особености (за да не се губи време):**
- Името на папката съдържа **безточково „ı"** (`Claude-Websıte`). Пътищата се извличат с glob, не се преписват на ръка.
- Ако промяна в CSS/JS „не се вижда" — първо проверете **кеша на браузъра** (сайтът се отваря през `file://`). Затова връзките към ресурсите имат `?v=…`; вдигнете стойността при нова промяна или натиснете Cmd+Shift+R.
- Заглавията ползват увеличено междуредие (`h1` 1.32, останалите 1.26), защото кирилските букви с опашки (**У, Ц, Д, Щ**) се застъпваха с бревето над **„Й"**. Не намалявайте под **1.20**.


## Как да стартирате локално

Нужен е само Python 3 (без build стъпка, без npm):

```bash
cd "path/to/Cloude-Websıte"
python3 serve.py
# отворете http://127.0.0.1:4173
```

Или просто отворете `index.html` директно в браузър (двоен клик).

### Контактната форма

В момента формата е **демонстрационна** (симулира изпращане от клиента, не праща реален имейл). За да заработи реално, свържете я с услуга като:
- **Formspree** — добавете `action="https://formspree.io/f/ВАШ_ID"` и `method="POST"` на `<form data-contact>` в `contact.html`, и премахнете `e.preventDefault()` логиката в `main.js`, или
- собствен бекенд / имейл API endpoint.

## Технически бележки

- **Достъпност**: контраст WCAG AA, видими focus states, `aria` етикети, поддръжка на `prefers-reduced-motion` (анимациите се изключват за потребители, които го предпочитат).
- **Производителност**: няма build; изображенията са с `loading="lazy"`, анимациите ползват само `transform`/`opacity`. Ако GSAP не се зареди, съдържанието остава видимо (graceful fallback).
- **Шрифтове**: Oswald (заглавия), Inter (текст), JetBrains Mono (етикети/числа) — с пълна кирилица.
- **Библиотеки** (от CDN): GSAP + ScrollTrigger. За офлайн работа ги свалете локално в `assets/js/`.
- **Карта**: вградена интерактивна OpenStreetMap карта (без API ключ), стилизирана в тъмната тема чрез CSS филтър, с маркер на адреса; бутон „Отвори в Google Maps" отваря локацията в нов таб.

### Защо не Tailwind?
Заявено беше „HTML + Tailwind + GSAP". Tailwind обаче изисква или build стъпка (нужен Node.js, който липсва на тази машина), или CDN версия, която показва предупреждение „не за продукция" и тежи ~400KB. Затова е използвана **ръчно написана CSS дизайн-система** — по-лека и зареждаща по-бързо на всякакви устройства, което беше основното изискване. Ако предпочитате Tailwind, сайтът може да се конвертира.
